# Axphi

这是一个 `Phigros` 音游的 **制谱器**

设计上, 采用时间轴和关键帧的方式, 仿 `AE` 等视频编辑软件的操作方式



## 使用

最新的提交不是太能使用, 有很多 bug

如果你想使用, 请转到 
 - `7fea886` (chore: 可用版本) 

该提交的版本勉强可以使用
